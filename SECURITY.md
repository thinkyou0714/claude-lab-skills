# セキュリティポリシー

## 脆弱性の報告

本リポジトリは思考OS（判断資産）の Markdown / Python スクリプト集であり、
実行時の外部依存はありません（標準ライブラリのみ）。

セキュリティ上の懸念（例: スクリプトの任意コード実行リスク、公開すべきでない
情報の混入、リンク先の安全性）を見つけた場合は、以下で報告してください。

- GitHub の **Security Advisories**（推奨）: リポジトリの「Security」→「Report a vulnerability」
- もしくは **Issue**（機微情報を含まない範囲で）

公開前のチェックは判断ゲート **GATE-3（公開・共有ゲート）**
（[`src/lab-core/rules/judgment-gates.md`](./src/lab-core/rules/judgment-gates.md)）に従い、
個人名・内部パス・私的情報の混入を確認してください。
